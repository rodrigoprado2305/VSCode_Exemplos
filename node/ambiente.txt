instalar o nvm para controle de versão do node, 
https://github.com/coreybutler/nvm-windows/releases

Baixar o nvm para Windows:

Acesse o repositório oficial do nvm para Windows no GitHub.
Baixe o arquivo nvm-setup.zip da versão mais recente.
Instalar o nvm:

Extraia o conteúdo do arquivo nvm-setup.zip.
Execute o arquivo nvm-setup.exe e siga as instruções do instalador para completar a instalação.
Instalar e usar a versão 20 do Node.js:

Abra um novo terminal (ou PowerShell) e execute os seguintes comandos:

nvm install 20
nvm use 20
node -v
